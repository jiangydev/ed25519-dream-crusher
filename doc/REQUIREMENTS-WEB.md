# 网页版 Pollard's Rho 攻击 Ed25519 项目

## 项目名称
Ed25519 Dream Crusher

## 项目概述
这是一个纯娱乐、教育性质的网页应用，让用户输入任意 Ed25519 公钥（32 字节 hex），启动 Pollard's Rho 算法尝试“破解”私钥。
核心目的是通过幽默的方式让用户直观感受到椭圆曲线离散对数问题（ECDLP）的经典攻击复杂度（≈2¹²⁶ 次操作），从而理解为什么 Ed25519 在经典计算机上是安全的。
整个计算完全在浏览器前端运行，无后端、无服务器依赖，便于开源并免费部署到 Cloudflare Pages。

## 核心功能要求

1. 固定目标曲线
   - 仅支持真实的 Ed25519 曲线（Curve25519，群阶 ≈ 2²⁵²）。
   - 不提供玩具小曲线选项（保持纯粹的“绝望感”和教育意义）。

2. 纯前端计算
   - 使用 JavaScript + BigInt 实现完整的 Pollard's Rho 算法。
   - 所有椭圆曲线运算（点加、点倍乘）使用成熟开源库实现（@noble/curves）。
   - 防浏览器崩溃方案（关键）：
     - 使用 requestAnimationFrame 或 setTimeout 分片执行，每次迭代循环最多执行 50,000 ~ 100,000 次操作，然后主动让出主线程。
     - 提供“速度控制滑块”（慢 / 中 / 快 / 极快），动态调整每帧迭代次数。
     - 实时监控 FPS，若检测到卡顿（< 10 FPS）自动降速并提示用户。
     - 提供“暂停 / 继续”按钮，用户可随时停止。

3. 持久化与分享功能
   - 运行状态（公钥、已执行步数、当前龟兔位置等）实时保存到 localStorage。
   - 页面刷新或关闭浏览器后重新打开可自动恢复。
   - 提供"一键生成分享链接"：将当前攻击状态编码为 URL 参数（使用 Base64 + gzip 压缩）。
   - 他人打开链接后自动恢复相同攻击状态（适合发给朋友一起"绝望"）。

4. 趣味性 UI 与实时日志
   - 整体风格：简洁、暗黑优先、带一点赛博朋克/复古终端感，使用 shadcn/ui 组件。
   - 页面结构：
     - 顶部：项目标题 + 幽默标语
     - 主区：居中的攻击控制卡片，包含：
       - 公钥输入框（带验证）
       - 开始 / 暂停 按钮
       - 当前步数计数器（科学计数法显示）
       - 实时速度（ops/sec）
       - 预计剩余时间（以"亿年"为单位，带夸张描述）
       - 滚动日志区（带表情、彩色文本）
     - 底部：速度控制滑块 + "生成分享链接"按钮
   - 日志示例（每 5 秒输出一次）：
      ```text
      🚀 已运行 1.2 × 10⁷ 步！龟龟和兔兔还在愉快散步～ 🐢🐇
      ⏱️ 当前速度：85,000 ops/sec（你的电脑很给力！）
      📅 预计还要等：约 4.7 × 10²⁸ 年（宇宙才 1.38 × 10¹⁰ 年，懂？😭）
      💡 小贴士：量子计算机还没出生，别急着等私钥哦～
      ```

5. 启动警告弹窗
   用户首次点击“开始攻击”时，弹出 Modal：
   ```text
   ⚠️ 严肃警告（但请笑着读）
   你正在对真实 Ed25519 公钥发起 Pollard's Rho 攻击
   理论需要约 2¹²⁶ 次操作
   按地球上最快电脑算，也要等 10³⁰ 年以上
   这纯粹是娱乐、教育、以及对数学难度的致敬
   不会成功，但会让你感受到“绝望”的美学 😂
   [我懂了，开始绝望之旅]   [先逃为敬]
   ```

## 技术栈
- 框架：Next.js 14（App Router） + TypeScript
- UI：Tailwind CSS + shadcn/ui（完全符合要求）
- 椭圆曲线库：@noble/curves（纯 JS、高性能、支持 ed25519）
- 状态管理：React Context / useState（轻量、单实例）
- 压缩/编码：pako（gzip） + Base64（用于分享链接）
- 部署：Cloudflare Pages（静态构建，免费、全球 CDN）
